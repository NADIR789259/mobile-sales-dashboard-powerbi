# 📊 Mobile Sales Data Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive mobile sales dashboard created using Microsoft Power BI.
The dashboard helps analyze mobile sales performance across cities, brands, payment methods, and time periods.

The main objective of this project is to transform raw sales data into meaningful business insights.

## 🎯 Project Objectives

* Analyze overall mobile sales performance
* Identify top performing cities and brands
* Understand customer purchasing behavior
* Track monthly sales trends
* Analyze payment methods used by customers

## 📈 Dashboard Features

### Key Performance Indicators (KPIs)

* Total Sales: **769.20M**
* Total Quantity Sold: **19K**
* Total Transactions: **4K**
* Average Sales Value: **40K**

### Sales by City

Interactive map visualization showing mobile sales distribution across different cities in India.

### Total Quantity by Month

Line chart displaying monthly sales quantity trends.

### Customer Ratings

Visualization showing customer satisfaction ratings.

### Transactions by Payment Method

Pie chart displaying payment methods:

* UPI
* Debit Card
* Cash
* Credit Card

### Brand Performance

Comparison of brands including:

* Apple
* OnePlus
* Samsung
* Vivo
* Xiaomi

Metrics analyzed:

* Total Sales
* Total Quantity
* Transactions

### Sales by Mobile Model

Bar chart showing top selling mobile models.

### Sales by Day Name

Line chart analyzing sales trends by day of the week.

## 🛠 Tools Used

* Microsoft Power BI
* Data Visualization
* Data Analysis

## 📊 Key Insights

* Some cities contribute significantly to total sales.
* Digital payments such as UPI and cards dominate transactions.
* Certain mobile brands generate higher sales compared to others.
* Monthly sales trends help identify demand patterns.
